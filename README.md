# Эхо Сердца — AI-анализатор ЭКГ

## 📊 О проекте
**Эхо Сердца** — программное обеспечение для анализа электрокардиограмм с использованием искусственного интеллекта. Проект направлен на создание вспомогательного инструмента для предварительной диагностики состояния сердца.

## 🎯 Цель проекта
Разработка приложения для анализа ЭКГ с помощью обученной модели ИИ, способного определять три состояния сердца:
* Нормальное сердцебиение
* Аномальное сердцебиение
* Инфаркт миокарда

## 🛠 Технологии
* **Python** — основной язык разработки
* **CustomTkinter** — создание пользовательского интерфейса
* **Ultralytics** — интеграция модели ИИ
* **YOLO** — алгоритм машинного обучения
* **Roboflow** — разметка данных

## 🔧 Установка
```bash
# Клонирование репозитория
git clone https://github.com/reamAdream/echo_heart

# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
python main.py
```

## 🎮 Использование
1. Запустите приложение
2. Нажмите кнопку «Выбрать файл»
3. Загрузите изображение ЭКГ
4. Дождитесь результатов анализа
